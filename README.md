# Golf-Statistics-Tracker

## Project Structure

```text
src/
├── Main.java // console menu and user interaction
├── GolfRound.java // represents a single golf round
├── GolfTracker.java // stores rounds and calculates statistics
└── FileManager.java // handles saving and loading data
```

## Example Menu

```text
==============================
   GOLF STATISTICS TRACKER
==============================

1. Add Round
2. View All Rounds
3. View Statistics
4. Delete Round
5. Search by Course
6. View Performance Trends
7. Save & Exit
```
## Features:

```text
Add new golf rounds
View all saved rounds
Delete existing rounds
Search rounds by course name
Calculate golf statistics, including:
Total rounds played
Best round
Worst round
Average score
Average putts
Average fairways hit
Average greens in regulation (GIR)
Average penalty strokes
Automatically save and load rounds using CSV file storage
```
